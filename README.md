## The Time-Traveling Queue
### Commands
ADD <val>: Enqueue. <BR/>
REMOVE: Dequeue. <BR/>
UNDO: Revert last op. <BR/>
PRINT: Show queue contents. <BR/>

## Buffered Keyboard Input Handler
### Commands
CONFIG <capacity> <process_rate>: Set buffer size and processing rate. <BR/>
KEY <char>: A keystroke arrives. <BR/>
BACKSPACE: Remove last unprocessed key. <BR/>
ENTER: Flush buffer to output. <BR/>
TICK: Process keys. <BR/>
OUTPUT: Show processed output. <BR/>
METRICS: Show drop rate and occupancy.

## The Sorted Contact Browser
### Commands
ADD <name> <phone>: Find spot, insert.  <BR/>
SCROLL_DOWN: Move cursor to next.  <BR/>
SCROLL_UP: Move cursor to prev.  <BR/>
DELETE_CURRENT  <BR/>
SHOW_CURRENT: Display details of selection.  <BR/>
